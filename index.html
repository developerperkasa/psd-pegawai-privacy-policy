Kebijakan Privasi PSD Pegawai

Terakhir diperbarui: 25 Agustus 2025

Tidak ada tracking lintas aplikasi Data hanya untuk fungsi aplikasi Cocok untuk semua umur

Ringkasan

Dokumen ini menjelaskan bagaimana aplikasi PSD Pegawai mengumpulkan, menggunakan, menyimpan, dan melindungi data Anda. Aplikasi ini TIDAK menggunakan tracking dan semua data hanya digunakan untuk fungsi internal aplikasi.

Dokumen ini disusun dengan merujuk pada ketentuan Undang-Undang No. 27 Tahun 2022 tentang Perlindungan Data Pribadi ("UU PDP").

Ringkasan Jenis Data

Jenis Data Keterkaitan Tujuan Tracking
Lokasi (Lokasi Presisi) Tertaut ke identitas Fungsi Aplikasi (verifikasi absensi) Tidak
Konten Pengguna (Foto/Video) Tertaut ke identitas Fungsi Aplikasi (foto absensi) Tidak
Konten Pengguna (Lainnya) Tertaut ke identitas Fungsi Aplikasi (jurnal, catatan) Tidak
User ID Tertaut ke identitas Fungsi Aplikasi (autentikasi, akun) Tidak

Semua data dikumpulkan hanya untuk fungsi internal aplikasi dan tidak digunakan untuk tracking atau iklan.

Rincian Pengumpulan & Penggunaan Data

1) Data yang Dikumpulkan

  • Lokasi Presisi – digunakan untuk verifikasi kehadiran saat absensi, memastikan kehadiran di lokasi yang benar.
  • Foto/Video – digunakan untuk foto selfie saat absensi sebagai bukti kehadiran.
  • Konten Pengguna Lainnya – seperti jurnal pembelajaran, catatan kesehatan, dan data administrasi santri.
  • User ID – untuk identifikasi unik pengguna dalam sistem aplikasi.

2) Tujuan Penggunaan Data

  • Fungsi Aplikasi – semua data hanya digunakan untuk menyediakan fitur-fitur aplikasi seperti absensi, jurnal pembelajaran, dan administrasi santri.
  • Verifikasi Kehadiran – lokasi dan foto digunakan untuk memverifikasi kehadiran yang sah.
  • Manajemen Akun – User ID digunakan untuk autentikasi dan manajemen sesi pengguna.

3) Tidak Ada Tracking

  • Aplikasi TIDAK menggunakan tracking lintas aplikasi atau situs.
  • Data TIDAK dibagikan dengan pihak ketiga untuk tujuan iklan atau analitik.
  • Data TIDAK digunakan untuk profil pengguna atau personalisasi iklan.

Dasar Hukum Pemrosesan

  • Persetujuan – untuk akses kamera dan lokasi yang diperlukan untuk fitur absensi.
  • Pelaksanaan Kontrak – untuk menyediakan fungsi inti aplikasi PSD Pegawai.
  • Kepentingan yang Sah – untuk keamanan dan verifikasi kehadiran yang sah.

Berbagi Data

Aplikasi PSD Pegawai TIDAK membagikan data dengan pihak ketiga. Data hanya disimpan dan diproses secara internal untuk:

  • Fungsi Aplikasi – penyimpanan data absensi, jurnal, dan administrasi.
  • Keamanan – verifikasi kehadiran dan pencegahan kecurangan.
  • Pelaporan – laporan internal untuk manajemen pesantren.

Penyimpanan & Retensi Data

  • Data akun dan absensi disimpan selama akun aktif dan hingga 24 bulan setelah penutupan.
  • Foto absensi disimpan untuk keperluan verifikasi dan audit internal.
  • Data log sistem disimpan hingga 12 bulan untuk keamanan dan troubleshooting.

Keamanan Data

Kami menerapkan langkah-langkah keamanan yang wajar, termasuk enkripsi data dalam perjalanan, kontrol akses berbasis peran, dan audit internal secara berkala. Data disimpan di server yang aman dan hanya dapat diakses oleh personel yang berwenang.

Hak Anda

Sesuai UU PDP, Anda berhak untuk:

  • Mendapatkan informasi mengenai pemrosesan data pribadi Anda.
  • Mengakses dan/atau memperoleh salinan data pribadi Anda.
  • Meminta perbaikan atau penghapusan data yang tidak akurat.
  • Menarik persetujuan kapan saja untuk pemrosesan berbasis persetujuan.
  • Menyampaikan pengaduan kepada otoritas yang berwenang.

Untuk menjalankan hak Anda, silakan hubungi kami melalui informasi kontak pada bagian Kontak.

Kesesuaian untuk Semua Umur

Aplikasi PSD Pegawai dirancang untuk digunakan oleh semua umur dan tidak mengandung konten dewasa. Aplikasi ini digunakan untuk:

  • Manajemen absensi dan administrasi santri
  • Jurnal pembelajaran dan catatan akademik
  • Administrasi pesantren dan kepegawaian
  • Semua fitur cocok untuk pengguna dari segala usia

Perubahan Kebijakan Privasi

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan material akan diberitahukan melalui aplikasi. Tanggal pembaruan terakhir tercantum di bagian atas halaman ini.
